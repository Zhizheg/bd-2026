# E1 Исследователь
- Имеет **РЕЙТИНГ**, рассчитываемый на основе результатов деятельности, кол-ва публикаций и мероприятий
- Организован по строгой иерархии
- Имеет бэкграунд (магистр, бакалавр, etc.)
- Выполняя работу, обязаны участвовать в конференциях с целью получения публикаций

# E2 Звание исследователя
- Некоторые исследователи имеют звания (доцент, профессор, лучший бакалавр, etc.)
# E3 Направление
- Закрепляется один **РУКОВОДИТЕЛЬ НАПРАВЛЕНИЯ** (Пользователь)
- Один и тот же руководитель может руководить несколькими направлениями
- Направления **МОГУТ БЫТЬ** иерархически структурированы
- В рамках направления исследователи выполняют работы (докторская диссертация, УИР, etc.)
# E4 Тип работы
- Связаны только с работами
- Докторская диссертация, УИР, etc.
# E5 Работа
- Имеет язык
- Может иметь несколько исследователей
# E6 Конференция
- Имеет необязательную ссылку на предыдущую конференцию в случае периодических конференций
- Определяется перечень **направлений**, по которым принимаются **работы**
- Могут быть платными или бесплатными (см. **E12 Платёжный документ**)
- По итогам появляются **публикации**
## E7 Сторонний автор
- Может иметь статьи (публикации) с нашими **исследователями**
- Может иметь труды без наших исследователей (см. **E16 Труды сторонних лиц**)
# E8 Статья (публикация)
- Имеет несколько **исследователей (также сторонних)**
- Имеет язык
- Имеет свой **Список цитирования**
# E9 Цитирование (список цитирования)
- Может содержать ссылки на **статьи**
- Имеет ссылку на одну родительскую **статью**
# E10 Мероприятие
- Могут принимать участие несколько **исследователей**, может объединять несколько **тем** (направлений)
# E11 Платёжный документ
- Содержит информацию о покрытии расходов **исследователя** в рамках участия в **конференции** (оплата участия, поездка)
- Оплата может производиться за счёт вышестоящего по иерархии исследователя либо за счёт личных средств
# E12 Язык
- Относится к **работам** и **статьям**
# E13 Труды сторонних лиц
- Связаны со **сторонними авторами**
# E14 Действие
- Содержит информацию о конкретном действии пользователя в системе, которое влияет на данные системы
- Может задействовать нескольких пользователей
- Имеет **тип**
# E15 Пользователь (сотрудник)
- Реализует разграничение прав доступа к системе
- Набор доступных **типов действий** определяется совокупностью назначенных ролей
- Может руководить направлениями
- Может иметь несколько **ролей**
- Выполняет действия в системе
# E16 Тип действия пользователя
- Содержит информацию о типе действия, выполняемого пользователями
- Отдельные операции требуют обязательного подтверждения пользователем с соответствующими правами
